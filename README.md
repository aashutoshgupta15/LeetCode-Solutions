# LeetCode-Solutions 🚀

Welcome to my repository of LeetCode problem solutions! This repo is a collection of my journey through Data Structures and Algorithms (DSA), focusing on clean code and optimized complexity.

## 📌 Overview
This repository contains my solutions to various LeetCode problems. Each solution is implemented with a focus on:
* **Efficiency:** Aiming for the best time and space complexity.
* **Readability:** Clear variable naming and helpful comments.
* **Documentation:** Brief explanations of the approach used.

## 🛠️ Tech Stack
* **Language:** C++, Python
* **Platform:** LeetCode

## 💡 How I Approach Problems
1. **Understand:** Read the constraints carefully.
2. **Brute Force:** Think of the simplest way to solve it.
3. **Optimize:** Use HashMaps, Pointers, or Sliding Windows to reduce complexity.
4. **Dry Run:** Trace the code with edge cases like empty arrays or single elements.
